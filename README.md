# BlueMoon Apartment Management System

## Overview
The BlueMoon Apartment Management System is a comprehensive solution designed for managing apartment complexes efficiently. It provides various functionalities that facilitate property management, tenant services, and maintenance tracking.  

## Features
- **Tenant Management:**  
  - Register new tenants  
  - Update tenant information  
  - View tenant list  

- **Property Management:**  
  - Add and remove properties  
  - Assign tenants to properties  
  - Track property statuses  

- **Maintenance Management:**  
  - Request maintenance services  
  - Assign maintenance requests to staff  
  - Track maintenance status

- **Financial Management:**  
  - Manage rent collection  
  - Generate financial reports  
  - Track expenses and profits

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ShibaCoder2005/BlueMoon_KTPM.git
   ```
2. Navigate to the project directory:
   ```bash
   cd BlueMoon_KTPM
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
- To start the application, run:
   ```bash
   npm start
   ```
